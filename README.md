页面包括中国地图 数据统计 省份排行榜 城市分布排行


通过鼠标滚轮可以放大地图

点击地图上的省份板块可以跳转对应省份下属城市

点击城市板块可以进行好友添加记录

根据城市中添加的好友数量 地图颜色会发生变化

好友数量越多 颜色越深 省级地图同理

The page includes a map of China, statistics, province rankings, and city distribution rankings.

You can zoom in on the map using the mouse wheel.

Clicking on a province on the map will take you to the corresponding city.

Clicking on a city allows you to add friends.

The map color changes based on the number of friends added to that city.

The more friends, the darker the color. The same applies to provincial maps.
